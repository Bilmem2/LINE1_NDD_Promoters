# Results

This directory contains output files generated by the analysis pipeline.

## Files

| File | Generated by | Description |
|---|---|---|
| `promoter_line1_counts_all.bed` | `03_bedtools_intersect.sh` | Genome-wide LINE-1 counts per promoter window |
| `intron_line1_counts_all.bed` | `03_bedtools_intersect.sh` | Genome-wide LINE-1 counts per intronic interval |
| `promoter_counts_<geneset>.tsv` | `04_gene_sets.py` | Promoter counts filtered by each gene set |
| `intron_counts_<geneset>.tsv` | `04_gene_sets.py` | Intron counts filtered by gene set |
| `promoter_statistics.tsv` | `05_statistics.py` | Summary table: occupancy, exact p-values, effect sizes |
| `intron_matched_pairs.tsv` | `05_statistics.py` | Length-matched NDD vs housekeeping gene pairs |
| `promoter_gc_cpg_all.tsv` | `analyze_gc_cpg.py` | Raw GC content and CpG O/E values per promoter |
| `gc_cpg_statistics.tsv` | `analyze_gc_cpg.py` | Sequence composition robustness statistics |
