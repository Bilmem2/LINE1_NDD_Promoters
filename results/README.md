# Results

This directory contains the final output files and statistical summaries generated by the analysis pipeline. 

> **Note on Reproducibility:** To maintain a lightweight repository, large intermediate `.bed` files (`intron_line1_counts_all.bed` and `promoter_line1_counts_all.bed`) are excluded. They can be fully reproduced by running the shell scripts provided in the main directory.

## Files and Directories

| File / Directory | Description |
| :--- | :--- |
| `promoter_counts_*.tsv` | Promoter-level LINE-1 counts filtered for specific gene sets (`housekeeping`, `hpo_adhd`, `hpo_seizure`, `seizure_adhd`, `sfari_syndromic`, `sfari_tier1`). |
| `intron_counts_*.tsv` | Intron-level LINE-1 counts used for intronic control analyses (`housekeeping`, `sfari_tier1`). |
| `intron_matched_pairs.tsv` | Length-matched NDD vs. housekeeping gene pairs used to control for gene length confounding. |
| `promoter_statistics.tsv` | **Main Summary Table:** Contains occupancy rates, exact Mann-Whitney U p-values, and Rank-biserial *r* effect sizes. |
| `ensg_to_symbol.tsv` | Dictionary file mapping Ensembl Gene IDs (ENSG) to standard HGNC gene symbols for readability. |

## Robustness Check

No significant differences were observed in GC content (p=0.3289) or CpG O/E ratios (p=0.9665) between NDD Tier 1 and Housekeeping promoters, confirming that the LINE-1 depletion is independent of primary sequence composition.
