# Results

This directory contains the final output files, statistical summaries, and visualizations generated by the analysis pipeline. 

> **Note on Reproducibility:** To maintain a lightweight repository, large intermediate `.bed` files (e.g., raw intersections) are excluded. They can be fully reproduced by running the shell scripts provided in the main directory.

## Files

| File | Description |
| :--- | :--- |
| `*_l1_counts.tsv` | Individual LINE-1 count tables for each specific gene set (e.g., `hpo_seizure_l1_counts.tsv`, `housekeeping_l1_counts.tsv`, etc.). |
| `all_l1_stats_summary.tsv` <br> `l1_counts_summary.tsv` <br> `gene_lists_l1_summary.tsv` | Summary tables compiling occupancy rates, total gene counts, and basic statistics across all datasets. |
| `*_stats.txt` | Raw statistical output logs (Mann-Whitney U exact p-values, Rank-biserial *r* effect sizes) for each NDD subset compared against housekeeping genes. |
| `intronic_length_matched_stats.txt` <br> `length_matched_l1_stats.txt` | Statistical results for the length-matched control analyses to address intronic and gene length confounding. |
| `ensg_to_symbol.tsv` | Dictionary file mapping Ensembl Gene IDs (ENSG) to standard HGNC gene symbols for readability. |
| `l1_density_plot.png` | Visual output showing the distribution/density of LINE-1 insertions. |

## Robustness Check

No significant differences were observed in GC content (p=0.3289) or CpG O/E ratios (p=0.9665) between NDD Tier 1 and Housekeeping promoters, confirming that the LINE-1 depletion is independent of primary sequence composition.
